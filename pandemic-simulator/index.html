PANDEMIC
SIMULATOR

You can install this game to your device as an app!
Just click on "add to home screen" (phone)
or click install (pc, laptop, etc.)

You will have a grid of blocks where each block represents a person

Green Block means healthy
Yellow Block means infected
Red Block means dead

So when someone gets infected, you have to tap that block asap else the person (block) dies

When no infected person is left you win and if everyone dies, you loose
